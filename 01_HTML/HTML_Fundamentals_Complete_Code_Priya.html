1. Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure of a webpage, while CSS adds styling and JavaScript adds interactivity.

Key Concepts:

  • HyperText: Text that contains links to other texts
  • Markup: System for annotating text to define structure
  • Language: Has syntax rules like any programming language

HTML Versions:

  • HTML 4.01 (1999)
  • XHTML (2000)
  • HTML5 (2014) - Current standard

2. HTML Document Structure

Basic HTML Template:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Visible content goes here -->
</body>
</html>

Explanation of Each Part:

  • <!DOCTYPE html>: Declares the document type and HTML version (must be first line)
  • <html>: Root element of an HTML page, lang attribute defines language
  • <head>: Contains metadata (title, character set, viewport, links to CSS/JS)
  • <body>: Contains all visible content (all user-facing elements)

Essential Meta Tags:

<head>
    <!-- Character encoding -->
    <meta charset="UTF-8">
    
    <!-- For responsive design -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- Page description for SEO -->
    <meta name="description" content="Learn HTML fundamentals">
    
    <!-- Keywords for SEO -->
    <meta name="keywords" content="HTML, CSS, JavaScript, Web Development">
    
    <!-- Author information -->
    <meta name="author" content="Your Name">
</head>

Element Purpose

<nav> Navigation links
<main> Main content of the document
<article> Independent, self-contained content
<section> Thematic grouping of content
<aside> Content tangentially related to main content
<footer> Footer for its nearest ancestor

11. HTML Best Practices

1. Document Structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Descriptive Title</title>
</head>
<body>
    <!-- Content -->
</body>
</html>

2. Use Semantic Elements:

Bad: <div id="header">

Good: <header>

3. Proper Heading Hierarchy:

Bad: <h1>Title</h1> <h3>Subtitle</h3> (Skipped h2)

Good: <h1>Title</h1> <h2>Subtitle</h2> <h3>Section</h3>

4. Image Accessibility:

Bad: <img src="photo.jpg">

Good: <img src="photo.jpg" alt="Two people working on laptops at a coffee shop">

5. Form Best Practices:

Bad: <input type="text">

Good: <label for="username">Username:</label> <input type="text" id="username" name="username">

13. Quiz

Multiple Choice Questions:

  1. What does HTML stand for?
    1. Hyper Text Markup Language
    2. High Tech Modern Language
    3. Hyper Transfer Markup Language
    4. Home Tool Markup Language
  2. Which element is used for the largest heading?
    1. <h6>
    2. <head>
    3. <h1>
    4. <heading>
  3. What is the correct HTML for creating a hyperlink?
    1. <a url="http://www.example.com">Example</a>
    2. <a href="http://www.example.com">Example</a>
    3. <a>http://www.example.com</a>
    4. <link>http://www.example.com</link>
  4. Which attribute is required for images?
    1. title
    2. src
    3. alt
    4. width

True or False:

  1. HTML is a programming language. (False)
  2. You should only use one <h1> per page. (True)
  3. <div> is a semantic element. (False)
  4. HTML comments start with <!-- and end with -->. (True)
  5. The <head> element contains visible content. (False)

Answers:

Multiple Choice: 1. a, 2. c, 3. b, 4. b (src is required, alt is highly recommended)
