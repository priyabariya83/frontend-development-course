# Pizza Paradise Restaurant Landing Page

A fully responsive restaurant landing page for a pizza restaurant, built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works on all device sizes from mobile to desktop
- **Interactive Menu**: Filter pizzas by category (All, Vegetarian, Meat Lovers, Specialty)
- **Shopping Cart**: Add/remove items, update quantities, and calculate totals
- **Special Offer Timer**: Countdown timer for limited-time offers
- **Smooth Animations**: CSS transitions and animations for better UX
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Newsletter Subscription**: Form with validation
- **Testimonials Section**: Customer reviews with ratings
- **Back to Top Button**: For easy navigation on long pages

## How to Use

1. Clone or download the project files
2. Open `index.html` in your browser
3. All images are placeholder paths - replace with your actual images
4. Customize colors, text, and content as needed

## Key JavaScript Features

1. **Menu Filtering**: Filter pizzas by category
2. **Shopping Cart**: Add/remove items, persist in localStorage
3. **Countdown Timer**: For special offers
4. **Mobile Navigation**: Toggle menu
5. **Smooth Scrolling**: For anchor links
6. **Form Validation**: For newsletter subscription
7. **Back to Top**: Scroll to top functionality

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #e74c3c;      /* Main brand color */
    --primary-dark: #c0392b;       /* Darker shade for hover */
    --secondary-color: #f39c12;    /* Secondary accent color */
    --dark-color: #2c3e50;         /* Dark text/background */
    --light-color: #ecf0f1;        /* Light background */
}
